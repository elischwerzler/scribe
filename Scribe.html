📜 Scribe — Claude Transcript Book

Turn your Claude Code sessions into a searchable, readable book.

Find out what you worked on yesterday, last week, last month.

1. Click 📂 Load Transcripts above.

2. Pick any .jsonl files from your .claude/projects/<project>/ folder. (You can select many at once.)

3. Sessions appear in the left sidebar by date. Click any to read.

4. Use the search bar to find a file path, a command, or anything you typed.

5. The right panel shows artifacts — files touched, commands run, commits made.

6. Click ⬇ Export Session to save a session as Markdown — great for devlog writing.

On Windows, transcripts live at:
C:\Users\<you>\.claude\projects\<project-hash>\*.jsonl
